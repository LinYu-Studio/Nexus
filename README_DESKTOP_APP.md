# LVMO_GAME - Windows 桌面应用程序打包指南

本指南将帮助您将 LVMO_GAME 网站转换为可在 Windows 上运行的桌面应用程序。

## 准备工作

在开始之前，请确保您的系统上已安装以下软件：

1. **Python 3.7+** - 可从 [Python 官方网站](https://www.python.org/downloads/) 下载
2. **pip** - Python 包管理器（通常随 Python 一起安装）

## 步骤 1: 安装必要的依赖项

首先，您需要安装打包 Windows 应用程序所需的依赖项。运行以下命令：

```bash
pip install -r requirements_desktop.txt
```

该文件包含以下主要依赖项：
- Flask>=2.0.0 - Web 框架
- pywebview>=4.0.0 - 将 Web 应用转换为桌面应用的工具
- pyinstaller>=6.0.0 - 用于打包 Python 应用程序

## 步骤 2: 检查核心文件

确保以下核心文件已正确配置：

1. **desktop_app.py** - 桌面应用程序的主入口，使用 pywebview 包装 Flask 应用
2. **desktop_app.spec** - PyInstaller 打包配置文件
3. **package_desktop_app.bat** - 简化打包过程的批处理文件

## 步骤 3: 运行打包脚本

您可以通过以下两种方式之一进行打包：

### 方法 A: 双击批处理文件（推荐）

1. 在文件资源管理器中找到 `package_desktop_app.bat`
2. 双击该文件运行
3. 打包过程将自动执行，等待完成

### 方法 B: 命令行执行

1. 打开命令提示符（CMD）
2. 导航到项目目录：
   ```cmd
   cd e:\Code\LVMO_GAME
   ```
3. 执行打包命令：
   ```cmd
   pyinstaller desktop_app.spec
   ```

## 步骤 4: 获取打包结果

成功打包后，您可以在以下位置找到生成的应用程序：

```
dist\LVMO_GAME\LVMO_GAME.exe
```

## 运行桌面应用程序

直接双击 `LVMO_GAME.exe` 文件即可启动应用程序。首次运行时，程序会在您的用户目录下创建一个 `.LVMO_GAME` 文件夹来存储应用数据。

## 注意事项

1. **数据存储** - 应用程序会将数据（游戏、用户信息等）保存在 `%USERPROFILE%\.LVMO_GAME` 文件夹中

2. **文件路径处理** - 应用程序已配置为在打包后正确处理文件路径，无论是作为独立可执行文件运行还是从源代码运行

3. **管理员权限** - 某些操作（如写入系统目录）可能需要管理员权限

4. **自定义图标** - 如果需要添加自定义应用图标，请编辑 `desktop_app.spec` 文件中的 `icon` 参数

## 常见问题解决

### 问题：应用程序无法启动或出现错误

**解决方法**：
- 检查是否已安装所有必要的依赖项
- 查看应用程序目录中的日志文件（如果有）
- 尝试以管理员身份运行应用程序

### 问题：某些功能无法正常工作

**解决方法**：
- 确保应用程序有足够的权限访问文件系统
- 检查 `.LVMO_GAME` 文件夹中的数据文件是否完整

## 技术说明

### 实现原理

1. **Flask 服务器** - 在后台线程中运行 Flask Web 服务器

2. **PyWebView** - 提供一个原生窗口来显示 Flask 应用程序

3. **PyInstaller** - 将 Python 代码和依赖项打包成单个可执行文件

4. **数据处理** - 确保应用程序在打包后能正确读取和写入数据文件

### 主要功能

- 游戏浏览和下载
- 用户账户管理
- 搜索功能
- 按下载次数排序的游戏列表
