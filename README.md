# Nexus

开源游戏分享平台 · Win11 Fluent Design

---

## 概览

Nexus 是一个面向独立开发者的游戏分享与分发平台，前端 UI 采用 Windows 11 Fluent Design 设计语言，主打云母质感、半透明丙烯酸效果与明暗双主题。

技术栈：Python Flask + Jinja2 + CSS3
数据层：JSON 文件存储，零依赖数据库
状态管理：Flask Session + localStorage

---

## 功能

### 玩家侧
- 游戏库 — 浏览、搜索、下载原创与转载游戏
- 详情页 — 双栏布局，系统配置、DLC、CDK 一站式展示
- 钱包系统 — 余额管理、充值申请、历史记录
- 站内信 — 用户间实时消息与通知系统
- 审核流 — 原创作品提交 → 管理员审核 → 自动上架

### 开发者侧
- 开发者平台 — 原创作品发布、项目管理、版本更新
- DLC 系统 — 为游戏发布扩展内容，支持独立定价与文件上传
- CDK 管理 — 生成 16 位激活密钥
- 开发者资料 — 工作室名称、简介、社交媒体展示
- 收益统计 — 下载量阶梯结算，每 10 次下载自动计费

### 管理侧
- 用户管理 — 审批、封禁、权限控制
- 审核系统 — 原创作品审核与自动上架
- 充值管理 — 用户充值申请审核与手动到账
- 内容管理 — 游戏上下架、原创标记管理

---

## 设计系统

- 主题：明暗双模式，自动根据时段切换
- 布局：侧边栏 220px + 内容区自适应伸缩
- 质感：Mica 云母背景 + Acrylic 丙烯酸毛玻璃
- 品牌色：#60cdff 天蓝渐变
- 响应式：≤900px 侧边栏折叠为图标模式
